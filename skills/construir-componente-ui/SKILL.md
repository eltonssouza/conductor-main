# Skill — construir_componente_ui

**Quando usar:** Para criar/ajustar um componente de interface.

**Passos:**
1. Definir estados (default, loading, erro, vazio, foco).
2. Estruturar HTML semântico e acessível.
3. Estilizar com layout robusto e responsivo.
4. Ligar dados com busca mínima necessária.
5. Testar acessibilidade (teclado, leitor de tela, contraste) e resumir.
