# Skill — automatizar_testes

**When to use:** To create or stabilize test automation.

**Steps:**
1. Choose the right level in the pyramid for each case.
2. Write tests against observable behavior.
3. Eliminate *flakiness* and *test smells*.
4. Integrate into CI as a fast *gate*.
5. Evaluate the suite by regressions caught.
