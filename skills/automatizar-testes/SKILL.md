# Skill — automatizar_testes

**Quando usar:** Para criar/estabilizar automação de testes.

**Passos:**
1. Definir o nível certo na pirâmide para cada caso.
2. Escrever testes de comportamento observável.
3. Eliminar *flakiness* e *test smells*.
4. Integrar ao CI como *gate* rápido.
5. Avaliar a suíte por regressões pegas.
